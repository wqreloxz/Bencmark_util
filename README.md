#  Bencmark_util 

**Bencmark_util** — это я консольная утилита на Python для бенчмарка твоего ПК. Работает на Windows, macOS и Linux. Тестирует железо и выдает результат в баллах/bencmark test of your pc.

## что умеет:
* **CPU Benchmark** — грузит проц математикой и проверяет скорость вычислений.
* **RAM Benchmark** — забивает оперативку данными и замеряет скорость чтения/записи.
* **Disk Benchmark** — проверяет твой SSD/HDD на чтение и запись тяжелых файлов.
* **System Info** — показывает всё железо: модель проца, ядра, потоки и объем оперативной памяти.
